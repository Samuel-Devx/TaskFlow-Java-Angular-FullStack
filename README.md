#  Task-Flow

> Aplicação de gerenciamento de tarefas com interface simples e backend robusto — organize suas atividades com eficiência.

---

## Sobre o Projeto

O **Task-Flow** é uma aplicação fullstack voltada para o gerenciamento de tarefas do dia a dia. Inspirada em ferramentas como Trello e Notion, ela permite criar, visualizar, atualizar e excluir tarefas de forma prática.

O projeto é dividido em dois módulos principais:

- **Task-backend** — API REST desenvolvida em Java , responsável por toda a lógica de negócio e persistência de dados.
- **Task-frontend** — Densenvolvido em Angular.js, responsável pela interface do sistema.

---

##  Funcionalidades

- Criação de tarefas com título e descrição
-  Listagem de todas as tarefas cadastradas
-  Edição de tarefas existentes
-  Exclusão de tarefas
- Atualização de status das tarefas
- Métricas de Tasks

---

## Estrutura de Dados

A API retorna as tarefas no seguinte formato:

```json
{
  "id": "a1b2c3d4",
  "title": "Estudar Go",
  "description": "Revisar goroutines e channels",
  "status": "em andamento",
}
```
